Chapter 1 About this book

This book was inspired by my own exploration of data visualization, which was often appreciated by my team leaders and members when working in my team project. The need to present data which can be understood intuitively but at the same time save space always challenges me to create plots and graphs in my head and do trials and errors searching on Github and Stackoverflow. Creating a chart using this strategy involves hours of learning and tens of trials. Therefore, I do not want to waste what I have learnt. I hope this book gives others inspiration on visualizing data for their own projects. The plots or charts included in this book are maybe unique and rarely used. Standard data visualization can be found in such books as Data Visualization in R, Data Analysis in R, or R for Researchers.

1.1 Prerequisites

In order that you can use this books and use eyecathing plots and graphs in this book, you will need to have some basic to intermediate skills in using R. Wait! Don’t go away just yet! I started learning R seven years after I completed my Master’s degree, and I enjoyed every process of it, and I know you will too. You don’t have to be a statistic person to learn R, and I wrote this book by keeping that in mind. To learn R, please read Dai’s online book hosted at https://bookdown.org/yundai09/rbootcamp-2025/. You can also consider watching some over 20 minute tutorial videos on YouTube if you are more of a visual-type person.

1.2 If you need help

For beginner of R, when something does not work or when an error message appears, you might be frustated. However, there are some places where you can seek for some helps. First, it is best to search on Google. If you have an error message, you can copy a short portion of this message and paste it on Google. Do not trust AI generated answer on Google unless you know what you are doing. If you need further help, you can create an account and ask questions in some knowledge sharing platforms such as Stackoverflow and CrossValidated. If your questions are also related to statistics with R, you can search in sthda at https://www.sthda.com. If you still have more question specifically for the codes in this book, send me an email. My details are publicly available at my university website.

1.3 The presentation of this book

This book consists to two chapter chapters, one for quantitative data visualization and the other for qualitative counterpart. Each chapter is divided into several sections, which consist of one topic to make it easier to find the type of charts or plots you are looking for. I try to use the data from my own project, publicly available dataset, or dummy data to create the chart. If the codes provided do not directly produce the chart, I still attach the resulting chart manually. Finally, I provide a sample of paper which uses the corresponding data visualization whenever it is available.

1.4 The updates and revisions

The book is written in present tense form because it is an ongoing project until I can no longer maintain it. I will keep updating and adding more charts or plots along the way. The idea can come from my self article writing or my colleagues or students, or from you. Therefore, if you have any ideas to consider, kindly drop me an email.
